# Ministe MD License

This is template to make a license file your project or game with minisite!

Open the pages directory to create a new page.

```sh
npm i
npm run generate
```

Run the command to generate the whole site.


## Use the 

If you want to use this template,

```sh
npx create-samengine-project
```

Run the command and then select the version and the template.
